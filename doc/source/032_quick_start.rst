=====================
Quick start examples
=====================


Interaction demo
----------------

Interactive field calculation and interaction demo. Uses HandyBeam core for calculations, and `pyqtgraph` for windows, events and visualisation

.. code-block:: bash

    python hello_world.py


Jupyter notebook demo
---------------------

Basic field visualisation and basic API usage demo. Uses matplotlib.pyplot, embedded in jupyter, for visualisation.

.. code-block:: bash

    jupyter Hello_world.ipynb
